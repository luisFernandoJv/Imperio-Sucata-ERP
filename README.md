### 🚀 O Projeto
O **Império Sucata ERP** é uma plataforma robusta projetada para transformar a operação bruta de reciclagens em um fluxo de dados inteligente. Focado em **Business Intelligence**, o sistema centraliza transações financeiras, controle de inventário e análises preditivas em uma interface moderna e resiliente.

---

### 💎 Diferenciais Estratégicos

| Módulo | Funcionalidade Principal | Valor Agregado |
| :--- | :--- | :--- |
| **BI & Analytics** | Dashboards reativos e KPIs em tempo real. | Decisões baseadas em lucratividade real. |
| **Inventário** | Sincronização Serverless via Cloud Functions. | Precisão total e alertas de ruptura de estoque. |
| **Financeiro** | Gestão de compras/vendas com exportação PDF. | Profissionalismo no fechamento de contas. |
| **Arquitetura** | PWA com suporte a modo offline. | Operação contínua mesmo sem estabilidade de rede. |

---

### 🛠️ Stack Tecnológica

| Tecnologia | Finalidade |
| :--- | :--- |
| **React 19** | Interface de usuário reativa e performática. |
| **Firebase** | Autenticação, Firestore e Serverless Functions. |
| **Tailwind CSS 4** | Design industrial escalável e responsivo. |
| **TanStack Query** | Gerenciamento de estado de servidor e cache. |
| **Recharts** | Visualização de dados e tendências de mercado. |

---

### 🔒 Segurança de Nível Corporativo
* **Isolamento de Credenciais**: Gestão rigorosa via variáveis de ambiente (`.env`).
* **Acesso Granular**: Regras de segurança (Security Rules) aplicadas a nível de documento no Firestore.
* **Autenticação Resiliente**: Sessões criptografadas com proteção contra ataques de força bruta.

---

### ⚡ Instalação Rápida

1.  **Clone o projeto:**
    ```bash
    git clone [https://github.com/luisFernandoJv/Imperio-Sucata-ERP.git](https://github.com/luisFernandoJv/Imperio-Sucata-ERP.git)
    ```
2.  **Configure as dependências:**
    ```bash
    cd sistema && npm install
    ```
3.  **Ambiente:** Adicione suas credenciais Firebase no arquivo `.env.local`.
4.  **Execute:**
    ```bash
    npm run dev
    ```
